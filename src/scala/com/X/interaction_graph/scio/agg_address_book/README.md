## InteractionGraphAddressBook Dataflow Job

#### IntelliJ
```
./bazel idea src/scala/com/X/interaction_graph/scio/agg_address_book:interaction_graph_address_book_scio
```

#### Compile
```
./bazel build src/scala/com/X/interaction_graph/scio/agg_address_book:interaction_graph_address_book_scio
```

#### Build Jar
```
./bazel bundle src/scala/com/X/interaction_graph/scio/agg_address_book:interaction_graph_address_book_scio
```

#### Run Scheduled Job
```
export PROJECTID=twttr-recos-ml-prod
export REGION=us-central1
export JOB_NAME=interaction-graph-address-book-dataflow

bin/d6w schedule \
  ${PROJECTID}/${REGION}/${JOB_NAME} \
  src/scala/com/X/interaction_graph/scio/agg_address_book/config.d6w \
  --bind=profile.user_name=cassowary \
  --bind=profile.project=${PROJECTID} \
  --bind=profile.region=${REGION} \
  --bind=profile.job_name=${JOB_NAME} \
  --bind=profile.environment=prod \
  --bind=profile.date=2022-04-13 \
  --bind=profile.output_path=processed/interaction_graph_agg_address_book_dataflow
```
